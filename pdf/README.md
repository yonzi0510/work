# PDF 변환기

PDF를 **Markdown·이미지**로 바꾸거나, **이미지·텍스트 파일을 PDF**로 만드는 도구입니다.

- 사용: [https://yonzi0510.github.io/work/pdf/](https://yonzi0510.github.io/work/pdf/) 접속
- **모든 처리는 브라우저 안에서** 이뤄지며 파일은 어디로도 전송되지 않습니다. 인터넷 없이도
  동작합니다(파일 하나로 실행).

## 세 가지 기능 (위쪽 탭)

1. **📝 PDF → Markdown** — PDF 글자를 Markdown 문서로 추출. 큰 글씨는 제목으로, 목록은
   목록으로 인식. 미리보기·복사·`.md` 다운로드 지원.
2. **🖼️ PDF → 이미지** — 페이지마다 이미지 한 장으로 저장(PNG/JPEG, 해상도 선택).
3. **📕 파일 → PDF** — 이미지(JPG·PNG)나 텍스트/Markdown(.txt·.md)을 PDF로 변환. 한글 지원.

## 참고

- 텍스트 기반 PDF에 최적화되어 있습니다. 스캔·사진으로 만든 PDF는 글자가 추출되지 않을 수
  있습니다(이럴 땐 이미지 변환을 쓰세요).
- 여러 파일을 한 번에 올리면 순서대로 처리됩니다.
