# 🎨 그림 학습지 생성기

어르신 인지훈련용 **그림 학습지**를 만들어서 인쇄하는 도구입니다.
지금은 1차로 **개수 세기** 학습지가 들어 있고, 도형 배열·줄 따라 색칠·숫자 색칠은 준비 중입니다.

👉 주소: https://yonzi0510.github.io/work/pg/draw/

인터넷이 없어도 파일을 열면 그대로 동작합니다. (글씨체만 기본 글씨체로 바뀝니다.)

## 개수 세기 학습지란?

종이 위에 참새·사과·항아리 같은 친숙한 그림이 흩어져 있고,
어르신이 그림을 세어서 아래 빈칸에 개수를 적는 학습지입니다.
세면서 그림에 ○표를 하시면 더 좋습니다.

## 사용 방법

1. **난이도를 고릅니다.**
   - 🌱 쉬움: 인지저하가 많으신 분 — 그림이 크고 2종류만 나와요
   - 🌟 보통: 기본 — 3~4종류가 알맞게 흩어져요
   - 🔥 어려움: 초기·경도이신 분 — 5~6종류가 촘촘하게 나오고, 세지 않는 "함정 그림"과
     "모두 몇 개?" 칸도 있어요
2. **[🔀 새 문제 만들기]** 를 누를 때마다 새로운 배치의 문제가 나옵니다.
   같은 난이도라도 매번 다른 학습지를 뽑을 수 있어요.
3. 오른쪽 미리보기가 **실제 인쇄 모습 그대로**입니다. 마음에 들면 **[🖨️ 인쇄하기]** 를 누르세요.
   (A4 세로, 여백 "없음"으로 인쇄하면 화면과 똑같이 나옵니다.)

## 고급 옵션

- **이름·날짜 칸 넣기**: 학습지 오른쪽 위의 이름·날짜 쓰는 칸을 넣거나 뺄 수 있어요. (기본: 넣기)
- **정답 보기**: 직원 확인용입니다. 켜면 답칸에 정답 숫자가 빨갛게 표시돼요.
  ⚠️ **켠 채로 인쇄하면 정답도 같이 인쇄되니**, 어르신께 드릴 종이는 꼭 끄고 인쇄하세요.

## 알아 두면 좋은 점

- 학습지에는 난이도 이름이 인쇄되지 않아요. 어르신이 보시는 종이에는 문제만 나옵니다.
- 그림은 절대 서로 겹치지 않게 배치돼요.
- 어려움 난이도의 "함정 그림"은 아래 답칸에 없는 그림이에요. 답칸에 있는 그림만 세시면 됩니다.
