# 숫자찾기 생성기

인지훈련용 **숫자찾기 학습지**를 무한 생성하는 도구입니다. 12×12 격자에 제시된 묶음이
모두 숨어 있습니다. 한 부 = 10장.

- 사용: [https://yonzi0510.github.io/work/pg/1/](https://yonzi0510.github.io/work/pg/1/) 접속
- 인터넷 없이도 동작합니다 (파일 하나로 실행).

## 사용 순서

1. **난이도(방향)** 선택 — 쉬움(가로·세로) / 보통(거꾸로 포함) / 어려움(대각선 포함)
2. **페이지별 스타일 구성** — 장마다 숫자만·한글·영어·기호·특수기호 지정
   (한 번에 적용, 난이도 점증 섞기 버튼 제공)
3. **총 장수** 고른 뒤 **생성** → **인쇄/PDF 저장**
4. 정답을 보려면 "정답 표시(격자에 음영)" 체크
