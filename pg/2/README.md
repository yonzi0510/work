# 단어찾기 생성기

인지훈련용 **단어찾기 학습지**를 만드는 도구입니다. 12×12 한글 격자에 주제별 단어를
숨겨 생성하며, 제시된 단어는 모두 격자 안에 들어 있습니다. 한 부 = 10장.

- 사용: [https://yonzi0510.github.io/work/pg/2/](https://yonzi0510.github.io/work/pg/2/) 접속
- 인터넷 없이도 동작합니다 (파일 하나로 실행).

## 사용 순서

1. **난이도(방향)**·**한 장당 단어 수** 선택
2. **페이지별 주제 구성** — 장마다 동물·과일·채소·음식·가족 등 주제 지정
   (무작위 섞기는 여러 주제를 섞어 한 장을 구성)
3. **총 장수** 고른 뒤 **생성** → **인쇄/PDF 저장**
4. 정답을 보려면 "정답 표시(격자에 음영)" 체크

Day:음 (dayeum.co.kr) 부속 도구.
