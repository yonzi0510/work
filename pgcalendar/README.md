# 강사배치 달력

주야간보호센터의 **월별 강사배치표(가감산 대비)**를 만드는 달력 도구입니다.

- 사용: [https://yonzi0510.github.io/work/pg2/](https://yonzi0510.github.io/work/pg2/) 접속 → 강사 추가 → 강사 선택 후 날짜 클릭
- 요일 제목(예: 금요일)을 클릭하면 그 요일 전체가 한 번에 채워집니다.
- 입력 내용은 사용하는 브라우저에 자동 저장되며, [백업 저장/불러오기]로 파일 백업도 가능합니다.
- 🖨 인쇄/PDF 저장(A4 가로), 🖼 PNG 이미지 저장 지원.
