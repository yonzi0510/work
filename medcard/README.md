# 투약정보 카드 생성기

투약기록지 PDF를 올리면, 어르신별 **투약정보 카드**(검색·질환 태그 필터)와 **엑셀 표**를 자동으로 만들어 주는 도구입니다.

- 사용: [https://yonzi0510.github.io/work/medcard/](https://yonzi0510.github.io/work/medcard/) 접속
- **모든 처리는 브라우저 안에서** 이뤄지며, 올린 PDF는 어디로도 전송되지 않습니다. 인터넷 없이도
  동작합니다(파일 하나로 실행).

## 사용법

1. 화면의 업로드 칸에 이번 달 투약기록지 PDF를 끌어다 놓거나, 눌러서 선택합니다.
2. 자동으로 분석되고, 어르신별 카드가 화면에 바로 나타납니다.
3. 검색창에 이름·질환·약품명을 입력하거나, 질환 태그 칩을 눌러 원하는 어르신만 걸러볼 수 있습니다.
4. **📊 엑셀 다운로드** — 번호·이름이 세로 병합된 표(`투약정보_최종.xlsx`)로 저장합니다.
5. **🗂 카드 페이지 다운로드** — 지금 화면의 카드 페이지를 독립된 HTML 파일(`투약정보카드.html`)로
   저장합니다. 이 파일도 인터넷 없이 열립니다.
6. 다른 달 PDF로 다시 만들려면 **↺ 새 PDF로 다시 만들기**를 누르세요.

## ⚠️ 확인이 필요한 항목

카드 위쪽에 자동으로 뜨는 안내 패널입니다. 프로그램이 스스로 판단하지 않고 사람 확인을 요청하는
내용이니 꼭 한 번 읽어보세요.

- **효과 표기 확인 필요** — 무코스타(위점막 보호 vs 위궤양 치료), 쿠에타핀·쿠티아핀·데파킨·
  리스페리돈 계열(조현병/양극성 vs 치매 행동증상 BPSD) 등은 어르신마다 처방 목적이 다를 수 있어
  자동으로 확정하지 않습니다. 카드에는 PDF 원문이 그대로 표시됩니다.
- **일부기간만 투약된 약품** — 전체 기록 기간 중 70% 미만으로만 나타난 약품입니다. 결석·처방
  중단·필요시(PRN) 투약 중 무엇인지는 사람이 확인해야 합니다.
- 이름 아래 질환 태그는 **처방된 약품 구성에서 자동 추정**한 것으로, 진단명 원본이 아닙니다.

## PDF 형식

삽교주간보호센터 형식의 "투약 기록지"(수급자별 메타정보 표 + 투약일자·약품명·투약량·투약방법·
투약시간 표 + 투약효능 표 반복 구조) PDF에 맞춰져 있습니다. 이 형식과 다른 PDF는 수급자 정보를
찾지 못했다는 안내가 뜰 수 있습니다.

## 알려진 제한

- 엑셀 파일은 병합 셀·표 구조는 정확하지만, 헤더 색상 같은 서식은 들어가지 않습니다(브라우저에서
  무료로 쓸 수 있는 엑셀 라이브러리의 제한).
- 표가 페이지 경계에서 예상과 다르게 잘리는 매우 드문 PDF 레이아웃에서는 일부 항목이 누락될 수
  있습니다. 카드 상단의 "수급자 ○명" 숫자와 원본 인원수가 다르면 원본 PDF를 확인해 주세요.
