# 이동서비스 안내서

주야간보호센터의 **이동서비스 변경 안내서(차량운행표)**를 만드는 도구입니다.
왼쪽에 입력하면 오른쪽 문서에 바로 반영되고, 인쇄·PDF·JPG로 저장할 수 있습니다.

- 사용: [https://yonzi0510.github.io/work/move/](https://yonzi0510.github.io/work/move/) 접속
- 인터넷 없이도 동작합니다 (파일을 내려받아 열어도 모든 기능 사용 가능).

## 사용 순서

1. **차량 목록 관리** — 쓰는 차량을 추가/삭제 (드롭다운의 "+ 직접 입력"으로도 등록)
2. **수급자 정보** — 이름·보호자·연락처·주소 입력
3. **변경 적용된 시간표 / 변경 전·후 내역** — 주중·토·일 시간과 차량 입력
   (토·일 미이용이면 체크를 풀면 해당 줄이 문서에서 사라집니다)
4. **서명부** — 기관명 입력, 직인 이미지 업로드 후 크기·위치 조절
5. **인쇄/PDF** 또는 **JPG 저장**

## 참고

- 직인 이미지를 넣기 전에는 직인이 표시되지 않습니다(깨진 그림 없이 빈 상태). 서명부에서
  직인 이미지를 업로드하면 나타납니다.
- 인쇄는 A4 세로 기준으로 맞춰져 있습니다.
