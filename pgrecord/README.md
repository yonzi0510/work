# 프로그램 일지 생성기

장기요양 주야간보호센터의 **프로그램 운영기록지 "반응 및 특이사항" 문장**을 만들어주는 도구입니다.

- 사용: [https://yonzi0510.github.io/work/pgrecord/](https://yonzi0510.github.io/work/pgrecord/) 접속 → 프로그램명·유형·키워드·인원 입력 → 생성
- 실제 운영기록지에서 추출한 문장 2,000여 개가 내장되어 있어 **인터넷 연결 없이도**(파일을 내려받아 열어도) 동작합니다.
- 문장은 ~하심/~하셨음 공문서체이며, 같은 회차 안에서 겹치지 않게 배분됩니다.

## 파일

| 파일 | 설명 |
|---|---|
| `index.html` | 생성기 본체 (문장은행 내장, 이 파일 하나로 동작) |
| `문장은행.json` | 문장 원본 데이터 (재빌드용 보관본 — 사용에는 불필요) |

Day:음 (dayeum.co.kr) 부속 도구.
