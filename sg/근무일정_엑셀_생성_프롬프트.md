# 근무일정 엑셀 생성 프롬프트

## 사용 방법
1. 아래 프롬프트를 복사해서 Claude에게 붙여넣기
2. `종사자목록_YYYYMM.xlsx` 파일을 첨부

---

## 프롬프트

```
첨부한 종사자목록_YYYYMM.xlsx 파일을 읽어서 직원 근무일정 엑셀을 만들어줘.

## 입력 파일 구조 (종사자목록)
- 직원 1명 × 31일 = 31행씩 반복
- 주요 컬럼: 성명, 생년월일, 근무기간, 직종, 일자(01~31), 요일, 근무시작, 근무종료, 주간휴식, 근무상세(근무/휴가)
- 시설장 등 일부 직원은 하루에 여러 행(시간대 분리)이 있음 → 해당 날짜의 첫 번째 주요 근무 행 기준으로 처리
- 근무상세가 "휴가"인 경우 → 휴가로 표시
- 근무시작/종료가 비어 있으면 → 휴무(비번)로 처리

## 출력 파일 형식
파일명: `YYYY년 MM월_직원근무일정.xlsx`

### 시트 구성
**시트 1: 직원근무일정**
- 1행: 헤더 (성명, 직종, 1일(요일), 2일(요일), ..., 31일(요일), 총근무일수, 총근무시간)
- 2행~: 직원 1명당 1행
- 각 날짜 셀 내용:
  - 근무: "HH:MM~HH:MM" (예: 08:20~17:40)
  - 휴가: "휴가"
  - 휴무: 공백

### 서식 규칙
- 폰트: 맑은 고딕, 10pt
- 헤더 행: 배경색 연한 파란색(4472C4), 흰 글씨, 굵게
- 토요일 열: 배경색 연한 파란색
- 일요일 열: 배경색 연한 빨간색
- 근무 셀: 흰 배경
- 휴가 셀: 배경색 연한 노란색
- 모든 셀 테두리: 얇은 선
- 열 너비: 성명(8), 직종(12), 날짜열(9), 합계열(10)
- 행 높이: 20

### 집계 컬럼
- 총근무일수: 해당 직원의 근무(공백 아닌 날 중 휴가 제외) 일수
- 총근무시간: 근무시간 합산 (분 단위 계산 후 시간으로 표시, 예: 45.5h)

## 주의사항
- 시설장(1.시설장)은 **육아기 단축근무 중**으로, 매달 근무 패턴이 동일하게 고정되어 있음
  - 근무일(월~금): 09:00~09:30 + 09:30~16:00(주간휴식 30분) + 16:00~17:30 → 표시: "09:00~17:30 (단축)"
  - 주말/공휴일: 휴무
  - 종사자목록 파일에서 시설장 데이터를 읽되, 단축근무 특성상 하루에 여러 행으로 분리되어 있음 → 해당 날짜 전체 시간을 합산해서 표시
- 요일은 실제 해당 월 날짜 기준으로 계산 (파일의 요일 컬럼 활용)
- 31일이 없는 달(예: 6월)은 해당 열 생략
- 파일은 `C:\Users\sapgy\Claude\Projects\근무표\` 폴더에 저장
```
