# 근무표(삽교)

삽교 센터의 **월별 근무 편성표**를 만드는 도구입니다. 자동 배치 후 손으로 고쳐서
엑셀·JPG로 내려받을 수 있습니다.

- 사용: [https://yonzi0510.github.io/work/sg/](https://yonzi0510.github.io/work/sg/) 접속
- 인터넷 없이도 동작합니다 (파일을 내려받아 열어도 모든 기능 사용 가능).

## 사용 순서

1. **연도·월 선택** 후 **⓪ 일요일 운영일 선택** — 운영하는 일요일만 체크
2. **① 사전입력** — 전 직원의 연차·교육·대휴·휴를 셀 클릭으로 입력 (입력한 칸은 🔒고정)
3. **② 자동 배치** — 규칙대로 나머지 차량·근무가 채워짐
4. 표에서 직접 수정 — 고친 칸은 🔒고정되어 ②를 다시 눌러도 유지 (더블클릭 = 고정 해제)
5. **배치표 엑셀 / JPG**, **③ 근무시간 엑셀**로 내려받기

## 그 밖의 기능

- 🌴 연차/대휴 정보: 발생·사용·잔여 확인과 수정 (소멸일 자동 계산)
- 📄 전체 안내 공문 / 📋 개인별 사용촉진: 연차 사용촉진 통지서 인쇄·JPG 저장
- 📝 휴무 신청표: 당월 일자가 채워진 신청표 인쇄
- 💾 작업 저장 / 📂 불러오기: 작업 내용을 파일(JSON)로 백업·복원
